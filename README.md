# Node.js Learning Path - Step by Step

Welcome! This is a complete guide to learning Node.js from basics to intermediate concepts.

## Curriculum Overview

### Phase 1: Fundamentals (Lessons 1-3)

1. **Lesson 1**: Node.js Basics & Global Objects
2. **Lesson 2**: Module System (require, exports)
3. **Lesson 3**: Working with Callbacks

### Phase 2: Async Programming (Lessons 4-6)

4. **Lesson 4**: Promises
5. **Lesson 5**: Async/Await
6. **Lesson 6**: Error Handling in Async Code

### Phase 3: Core APIs (Lessons 7-9)

7. **Lesson 7**: File System Operations
8. **Lesson 8**: Path Module & Stream Basics
9. **Lesson 9**: Event Emitter Pattern

### Phase 4: Servers & APIs (Lessons 10-12)

10. **Lesson 10**: HTTP Server Fundamentals
11. **Lesson 11**: Building a REST API
12. **Lesson 12**: Environment Variables & Configuration

## How to Use This Course

Each lesson includes:

- **Concept explanation** in comments
- **Working code examples**
- **Hands-on exercises** to practice

Run each lesson file:

```bash
node lesson-1-basics.js
node lesson-2-modules.js
# ... and so on
```

## Prerequisites

- Basic JavaScript knowledge (variables, functions, objects)
- Node.js installed (v14 or later)
- A text editor (VS Code recommended)
